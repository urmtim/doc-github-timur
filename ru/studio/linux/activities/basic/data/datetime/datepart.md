Компонент, выделяющий из даты нужную часть.

| Свойство    | Тип      | Описание                    |
| ----------- | -------- | --------------------------- |
| Дата\*      | DateTime | Исходная дата               |
| Часть\*     |          | Заданная часть даты/времени |
| Результат\* | Int32    | Результирующая часть        |
