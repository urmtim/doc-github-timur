Компонент, преобразующий строку к дате.

| Свойство    | Тип      | Описание                                      |
| ----------- | -------- | --------------------------------------------- |
| Строка\*    | String   | Исходная строка                               |
| Формат\*    | String   | Формат преобразования ("yyyy.MM.dd HH:mm.ss") |
| Результат\* | DateTime | Результирующая дата                           |
