Компонент, преобразующий дату к строке.

| Свойство    | Тип      | Описание                                      |
| ----------- | -------- | --------------------------------------------- |
| Дата\*      | DateTime | Исходная дата                                 |
| Формат\*    | String   | Формат преобразования ("yyyy.MM.dd HH:mm.ss") |
| Результат\* | String   | Результирующая строка                         |
